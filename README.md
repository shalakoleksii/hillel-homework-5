# Homework 3

## Опис

Сторінка з прикладами CSS-селекторів і розрахунком їх специфічності.

## Реалізовано

- теговий селектор
- селектор класу
- селектор id
- селектор атрибута
- вкладені селектори
- дочірні селектори
- послідовність селекторів
- різні комбінації селекторів тег/клас/атрибут/id/вкладеності

## Структура

- `index.html` — основна сторінка
- `style.css` — стилі

## Запуск

1. Відкрити файл `index.html` у браузері.
