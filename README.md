# Homework 3

## Опис

У цьому проєкті реалізовано багатосторінковий сайт з єдиним стилем оформлення, уніфікованим контейнером, структурованими формами та впорядкованою галереєю.

## Основні вимоги, виконані в проєкті

- Усі сторінки містять контент у контейнері з обмеженою шириною.
- Контейнер має внутрішні відступи та центрування на сторінці.
- Спільні елементи сайту, зокрема меню та футер, мають однаковий стиль на всіх сторінках.
- Форми структуровані, поля вирівняні та мають зручні відступи.
- Форми не притискаються до країв контейнера та займають розумну ширину.
- Галерея організована логічно, зображення розміщені по три елементи в ряд.
- Для стилізації використовуються теги, атрибути та класи.
- ID не використовуються для стилізації.
- Код HTML і CSS відформатований для зручного читання.
- Розмітка та стилі перевірені на валідність.

## Структура проєкту

- `index.html` — головна сторінка
- `style.css` — основні стилі сайту
- `pages/about.html` — сторінка "Про нас"
- `pages/articles.html` — сторінка блогу
- `pages/gallery.html` — сторінка галереї
- `pages/contacts.html` — сторінка контактів
- `pages/registration.html` — сторінка реєстрації

## Використані технології

- HTML5
- CSS3

## Перевірка валідності

Для перевірки коду використовувалися:

- [HTML Validator](https://validator.w3.org/#validate_by_upload)
- [CSS Validator](https://jigsaw.w3.org/css-validator/#validate_by_upload)

1. Завантажити або відкрити папку проєкту.
2. Відкрити файл `index.html` у браузері.
